# **App Name**: FOODIE

## Core Features:

- User Authentication: Separate login interfaces for users, owners, and delivery agents.
- Real-time Order Tracking: Track food orders in real time, displaying location of delivery agent and estimated time of arrival.
- Restaurant Management: Allow owners to manage menus, set availability, and view order history.
- Delivery Agent Interface: Enable delivery agents to view assigned orders, update delivery status, and optimize routes using Google Maps integration.
- Online Payment: Integrate Razorpay for secure online payments.
- Social Login: Enable Google sign-in option for quick and easy access.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7800) to convey energy and appetite.
- Background color: Light red (#F2F2F2), creating a clean and modern backdrop.
- Accent color: Complementary orange (#3498DB) for highlighting important interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern yet welcoming feel for easy readability.
- Use clean, outline-style icons for menu items, delivery status, and user actions.
- Design a card-based layout for menu items and restaurant listings for clear organization and ease of navigation.
- Incorporate subtle animations on order updates and map movements for a smooth user experience.
